# Activity Logs

Audit user and system activity captured via Spatie Activitylog.

## Table tools
- Badge column for log name plus truncated description and model type
- Searchable user column (`causer.name`) and sortable timestamp
- Filters for log name and date range (from/until)
- View action opens read-only detail panel; bulk delete available for cleanup

## Record view
- Shows subject/causer type & IDs
- Displays new vs. old values via key/value viewers
- Entries are read-only to preserve audit integrity
