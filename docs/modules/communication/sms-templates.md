# SMS Templates

Edit predefined SMS templates used for transactional messages.

## Form highlights
- Name and slug are locked to protect system references
- Subject field (optional) plus editable message body
- Placeholder list is view-only for reference

## Table tools
- Searchable name and copyable slug
- Filter by active/inactive flag
- Quick edit action; row links stay disabled to avoid accidental opens

## Notes
- Keep messages concise (ideally under 160 characters)
- Provide opt-out instructions when required by local regulation
